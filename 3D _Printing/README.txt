Printing Guide

打印指南
===========
Slicer files are made with Bambu Studio for Bambulab devices. Case can be printed in X series, P series and A1. Keycaps can be printed in all the devices.
For non-Bambulab devices, create a new project and drag all the single .3mf files into the slicer software. Maintaining 0.2mm layer height and 1 skirt.

切片文件使用Bambu Studio生成，仅适用于拓竹机器。外壳可以使用X1，P1和A1打印，键帽可以使用所有机器打印。
如果您使用非拓竹机器，打开切片软件新建项目，将所有的模型文件拖入项目，保持0.2mm层高的的默认工艺即可打印。添加一圈裙边可以有效改善首层瑕疵。
